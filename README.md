### `Github Action`을 사용한 CI/CD 시연 프로젝트
- `Github Flow` 적용
- `Dockerfile`을 사용한 이미지 빌드 & 배포
- 각 서버별 이미지 업데이트 후 컨테이너 자동 재실행(`watchtower` 사용)
  + 테스트 서버 30초 단위 이미지 갱신 체크
  + 운영서버 24시간 단위 이미지 갱신 체크
